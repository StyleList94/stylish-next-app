# Stylish Next App

Create stylish NEXT app

## Feature

### App Router

- `src` Directory
- Module path aliases: `@/` 
- Support TypeScript

### Testing

- Vitest
- DOM Testing with React Testing Library

### Code Formater

- ESLint Flat config
- Prettier
- Support `lint-staged`

### Styling

- `tailwindcss`

## Getting Started

```bash
pnpm create next-app --example https://github.com/StyleList94/stylish-next-app
```
